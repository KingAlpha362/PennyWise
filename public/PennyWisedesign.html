PennyWise design

Fonts — your current font is doing the job but it's not doing your brand any favours. My top recommendation is Plus Jakarta Sans for headings paired with Inter for body text. Jakarta Sans has wide, confident strokes that feel authoritative for a finance product — think the kind of energy Stripe or Wise give off. Both are free on Google Fonts and load in one line. If you want something more editorial/premium, the serif + sans split (Option B) works beautifully and would make PennyWise feel less like a typical SaaS and more like a trusted financial platform.

Light mode — the current scheme isn't bad, it's just a bit personality-free. Option A (crisp white + forest green) is what I'd go with — it pairs naturally with a dark mode that also uses green/teal accents, giving you a coherent dual-theme system rather than two themes that feel unrelated. The most important thing to fix regardless of colour choice is that "Learn more" secondary button — it's nearly invisible right now.

Dark mode — still recommend Option A (navy + teal). If you go with the green light mode, Option C (true black + emerald) becomes a great match since you'd stay in the green family across both modes. Option D (slate + cyan) is worth considering if you want something that feels a bit more "data dashboard" and less "marketing site."

Here's your complete theme file. Here's how to use it:

Drop it in — link it in your <head> after your existing stylesheet so it layers on top:

<link rel="stylesheet" href="pennywise-theme.css">

Dark mode toggle — the file uses both [data-theme="dark"] and prefers-color-scheme: dark, so it works automatically with the OS setting. If you have a manual toggle button, just do:

document.documentElement.setAttribute('data-theme', 'dark'); // or 'light'

Sticky nav — uncomment the JS snippet at the bottom of the file (section 7) and make sure your nav has the class navbar. It'll gain a background as the user scrolls past 20px.

Class names — the file assumes standard class names like .btn-primary, .btn-secondary, .hero-title, .pricing-card--pro etc. If yours differ, just do a quick find-replace on those names.

The biggest immediate wins you'll see just from this file: the secondary button becoming visible, the stat numbers popping in brand green/teal, the Pro pricing card getting a proper "Most popular" badge, and the hero mockup image getting a grounding shadow in both modes.
